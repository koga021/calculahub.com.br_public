Como funciona a Curva de Crescimento da OMS?

A Organização Mundial da Saúde (OMS) estabeleceu em 2006 os padrões internacionais de crescimento infantil. Diferente de tabelas antigas que apenas descreviam como as crianças cresciam, este padrão descreve como as crianças devem crescer em condições ideais de saúde e nutrição.

O Método LMS e o Z-Score

Esta calculadora utiliza o método estatístico LMS (Lambda, Mu, Sigma) para calcular o Z-Score. O Z-Score é a medida mais precisa para avaliar o crescimento, pois considera a distribuição estatística da população.

  • Peso para Idade: Avalia se o peso está adequado para a idade cronológica. Importante para detectar baixo peso ou excesso.
  • Comprimento para Idade: Reflete o crescimento linear acumulado. Baixa estatura pode indicar desnutrição crônica.
  • Perímetro Cefálico: Fundamental nos primeiros 2 anos para acompanhar o desenvolvimento neurológico.

Atenção aos Prematuros

Para bebês nascidos com menos de 37 semanas, é essencial utilizar a Idade Corrigida até os 2 anos de idade. Nossa calculadora faz esse ajuste automaticamente, garantindo que você não compare seu bebê prematuro com tabelas de nascidos a termo de forma injusta.
